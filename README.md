[jsonx][] is a pure JavaScript library which allows simple transformation
between XML and JSON.

## JSON to XML

```
JSONX.build(Object[]|String)
```

## XML to JSON

```
JSONX.parse(Node|Node[]|NodeList|String)
```

## Stringify

```
JSONX.stringify(Node|Node[]|NodeList|String)
```
